# CRIC-INFO-WEB-SCRAPPER
this is a web scrapping project which scraps data from the espnCricInfo.com website

npm modules to be installed from terminal.

<!-- xlsx , cheerio , postman-request -->

```
$ npm i xlsx
$ npm i cheerio
$ npm i postman-request
```
